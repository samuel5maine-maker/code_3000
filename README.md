Personal Ethics Statement:
When evaluating my own ethics on the use of Artificial Intelligence in the workplace. I stress the importance of Responsibility
as the core value. AI is a tool just like anything else it can do good it can do bad. I see no problem in leveraging it to do
good, however just you are responsibile for the good it can do, you are just as responsibile in the event it does bad. Perhaps, because of this emphasis on Responisibility, I believe AI usage should be as transparent and open as possible. If you are to use it
it should not be hidden, it should be documented, and openly told. Consumers should not have to be decieved by what is and isn't
AI and we can never learn what situtations AI is well suited for and what situations it isn't if both it's successes and failures
aren't documented. Finally, AI should not be considered a standalone artifact. I've never found the idea of AI as a substituion for
humans very palatable. Like I said, AI is a tool, a powerful tool, but still a tool it should be used in conjuction with at the 
absolute bare minimum human monitoring and manual confirmation of correctness. Never as an outright substitution.
